# Change Log

All notable changes to the "Dart JSON Generator" extension will be documented in this file.

## [1.0.0] - Initial Release

### Added
- Right-click context menu integration for Dart files and folders
- Progress notification during code generation
- Automatic file explorer refresh after generation
- Error handling with detailed output channel
- Support for both file and folder selection
- Cross-platform command detection (Windows, macOS, Linux)
